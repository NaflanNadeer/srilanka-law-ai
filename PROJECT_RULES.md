# Sri Lanka Law AI - Project Rules

## Goal

Build the most accurate AI assistant for Sri Lankan law.

## Principles

- Accuracy is more important than speed.
- Never invent legal citations.
- Every legal answer should reference its source whenever possible.
- Keep the code modular and well documented.
- Commit meaningful changes to Git frequently.
- Security and privacy should be considered from the start.
- Build features that scale to thousands of users.

## Coding Standards

- Use TypeScript.
- Keep components reusable.
- Avoid unnecessary dependencies.
- Use CSS Modules for styling.
- Keep functions focused on a single responsibility.
